# 🌐 LinguaX - Language Translator

LinguaX is a multilingual language translation web application built using Python, Streamlit, and Deep Translator.

The application allows users to instantly translate text between 14 supported languages through a clean and user-friendly interface.

## Features

* 🌍 Translate text across 14 languages
* 🇬🇧 English
* 🇫🇷 French
* 🇪🇸 Spanish
* 🇩🇪 German
* 🇮🇳 Hindi
* 🇮🇳 Tamil
* 🇯🇵 Japanese
* 🇨🇳 Chinese (Simplified)
* 🇹🇼 Chinese (Traditional)
* 🇰🇷 Korean
* 🇦🇪 Arabic
* 🇷🇺 Russian
* 🇵🇹 Portuguese
* 🇮🇹 Italian

### Additional Features

* Language selection with country flags
* Input validation
* Translation progress spinner
* Success and error notifications
* Translation summary metrics
* Responsive Streamlit interface

## Technologies Used

* Python
* Streamlit
* Deep Translator

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/LinguaX.git
cd LinguaX
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
